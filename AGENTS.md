## Conventions

- Site-visible text (nav, page titles, date format) is Traditional Chinese; code comments in `eleventy.config.mjs` and `input.css` are also zh-TW â€” keep both when editing.
- Global site metadata lives in `src/_data/site.json`; templates read it as `{{ site.* }}` â€” never hardcode the title, URL, or author in templates.
- Post filenames follow `YYYY-MM-DD-slug.md`; the date prefix must match the frontmatter `date` (enforced by tests/content.test.mjs).
- Tags that slugify to the same URL (`Docker`/`docker`, homophonous Chinese words) or to an empty string fail the build â€” rename the tag (source: lib/filters.mjs `assertNoSlugCollisions`).
